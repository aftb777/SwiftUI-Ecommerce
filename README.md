# SwiftUI-Ecommerce

A modern e-commerce application built with SwiftUI. This repository demonstrates a production-style shopping app UI with key features such as product browsing, detail screens, shopping cart, and simple checkout workflow.

## Features

- **Beautiful SwiftUI Interface:** Leveraging native iOS UI components for smooth user experience.
- **Product List & Detail Screens:** Browse products in a grid/list and view full details.
- **Shopping Cart:** Add, remove, and update items in the cart with dynamic updates.
- **Minimal State Management:** SwiftUI state management principles for scalable design.
- **Modular Code Structure:** Easily extensible architecture for future features and business logic.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/aftb777/SwiftUI-Ecommerce.git
```

Open the project in Xcode and build/run on an iOS simulator or device.

## Requirements

- Xcode 13 or later
- iOS 15.0 or later

## Folder Structure

- `Models` – Data models representing products and cart items.
- `Views` – All SwiftUI views/screens and reusable UI components.
- `ViewModels` – Classes managing data flow and business logic.

## How to Use

1. Launch the app.
2. Browse the home screen for available products.
3. Tap any product to see its details.
4. Add items to your cart and proceed to checkout.
